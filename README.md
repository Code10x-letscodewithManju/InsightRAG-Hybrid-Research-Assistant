# InsightRAG

**InsightRAG** is a Hybrid Retrieval-Augmented Generation (Hybrid RAG) research assistant that provides grounded, citation-backed answers from uploaded documents.

## Features (Planned)

- Hybrid Retrieval (Semantic + BM25)
- FAISS Vector Search
- Cross-Encoder Reranking
- Grounded Answer Generation
- Source Citations
- Confidence Score
- Evidence Panel
- Multi-PDF Support
- Streamlit Interface

## Tech Stack

- Python
- Streamlit
- LangChain
- Groq (Llama 3.3 70B)
- FAISS
- BM25
- Sentence Transformers
- PyMuPDF
